What's the purpose of this font?

Previously, no fonts for Omiega could be found on the Internet, so everything in Omiega had to be written on paper, which is inconvenient. The creation of this ttf font can extend the usage of Omiega and spread the system to others. If you happen to be watching this page, please download the font and help us achieve the goal!
